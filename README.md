# Mail Tuner
A tool to manage email configurations.

## Usage
1. Create a new instance of the `MailTuner` class.
2. Add email configurations using the `add_configuration` method.
3. Update email configurations using the `update_configuration` method.
4. Get all email configurations using the `get_configurations` method.
5. Get a specific email configuration using the `get_configuration` method.
6. Save email configurations to a file using the `save_configurations` method.
7. Load email configurations from a file using the `load_configurations` method.
